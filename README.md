# Thiranex-Project
Task 1: 
Password Strength Analyzer

Objective: Develop a tool to evaluate password complexity and prevent weak credential usage.
Key Features: Checks for length, character diversity (uppercase, lowercase, digits, symbols), and provides a secure alternative.
Security Highlight: Used the `secrets` library instead of `random` to ensure cryptographically secure password generation.

## Task 2:
Vulnerability Scanner

Objective: Identify open ports and potential service-related risks on a target host.
Key Features: Scans common ports (FTP, SSH, HTTP, etc.) and provides risk warnings.
Security Highlight: Focuses on reconnaissance, identifying unencrypted protocols like Telnet (23) or FTP (21).

## Task 3:
Phishing Email Detection Model

Objective: Build a machine learning classifier to distinguish between legitimate and malicious emails.
Key Features: Uses TF-IDF Vectorization and a Naive Bayes algorithm.
Security Highlight: Achieved high accuracy in detecting common phishing keywords and suspicious URL patterns.

## Task 4:
Secure Login System

Objective: Build a web-based authentication system resilient to common web attacks.
Security Implementations:
  - Password Hashing: Implemented salted hashing via `Flask-Bcrypt`.
  - SQLi Protection: Utilized SQLAlchemy ORM to prevent SQL Injection.
  - Session Management: Securely managed user states and logout functionality.
